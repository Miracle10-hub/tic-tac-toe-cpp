ATM Simulator - Student Project
---------------------------------
This is a simple ATM simulator created in C++ for portfolio purposes.
Features:
- Check balance
- Deposit money
- Withdraw money
- PIN verification

To run:
1. Open a C++ IDE or use g++ to compile:
   g++ atm_simulator.cpp -o atm_simulator
2. Run the program:
   ./atm_simulator (Linux/Mac) or atm_simulator.exe (Windows)
