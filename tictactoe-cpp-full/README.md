- Two-player (X vs O) mode
- Play vs Computer (simple AI: win/block/center/corner/random)
- Terminal interface
- Replay games easily

## How to run

Make sure you have **g++** installed.

```bash
git clone https://github.com/your-username/tic-tac-toe-cpp.git
cd tic-tac-toe-cpp
make
./tictactoe
```

Or compile manually:

```bash
g++ src/tictactoe.cpp -o tictactoe
./tictactoe
```
